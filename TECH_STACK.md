# Tech Stack

## Backend & Cloud Infrastructure
- **Firebase Realtime Database** - Cloud data synchronization and real-time message broker
- **HTTPS/TLS 1.2+** - Secure communication protocol
- **HTTP/1.1 REST API** - Application layer communication

## Computer Vision & Image Processing
- **Python 3.7+** - Primary development language
- **OpenCV (cv2)** - Real-time parking slot detection and occupancy analysis
- **NumPy** - Numerical computations for image processing

## IoT & Embedded Systems
- **ESP8266 Microcontroller** - Wi-Fi-enabled distributed control node
- **Arduino IDE / PlatformIO** - Firmware development and deployment
- **IEEE 802.11 b/g/n Wi-Fi** - Network connectivity

## Hardware Components
- **USB Webcam** - Video input source
- **Servo Motor** - Barrier actuation
- **Local Wi-Fi Router** - Network infrastructure

## Network & Protocols
- **TCP/IP Stack** - Foundation protocol suite
- **DHCP** - Dynamic IP addressing
- **NAT Gateway** - Network address translation
- **DNS** - Hostname resolution
- **JSON** - Data serialization format

## Development & Configuration
- **Python Virtual Environment (venv)** - Dependency isolation
- **Firebase Configuration (config.json)** - Credentials and database URL management
- **Git** - Version control

---

## Architecture Overview

This technology stack demonstrates a complete IoT system spanning the **OSI/TCP-IP reference models**, from physical Wi-Fi connectivity to cloud-based application logic, making it an excellent educational platform for networking and distributed systems concepts.

### Key Features:
- **Distributed Control**: ESP8266 nodes communicate via Firebase Cloud Database
- **Real-Time Updates**: Computer vision pipeline detects parking slot occupancy
- **Standard Protocols**: HTTP/HTTPS over TCP/IP for production-grade deployment
- **Scalable Design**: Modular architecture supporting multiple cameras and control nodes
